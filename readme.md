# XrayVPN

Описание
-------
XrayVPN — набор скриптов и конфигураций для быстрой установки и развёртывания Xray (VLESS + Reality) VPN-сервера. Репозиторий содержит сборочные скрипты, Docker-конфигурацию и инструкции для развёртывания на VPS.

Содержимое репозитория
----------------------
- `build-linux.sh`, `build-windows.ps1` — скрипты сборки и упаковки.
- `Dockerfile`, `docker-entrypoint.sh` — контейнеризация приложения.
- `deploy-vps.sh`, `full_setup.sh` — примеры автоматизированного развёртывания.
- `setup-xray-vless-reality.md` — пошаговая инструкция настройки и запуска Xray.

Быстрый старт
------------
1. Ознакомьтесь с инструкцией: `setup-xray-vless-reality.md`.
2. Для локальной сборки запустите на Linux: `./build-linux.sh` или на Windows: `./build-windows.ps1`.
3. Для деплоя на удалённый сервер используйте: `./deploy-vps.sh`.
